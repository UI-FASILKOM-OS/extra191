Prerequisites

Program yang perlu dipersiapkan untuk mengikuti perkuliahan.

  • Catatan: fitur SSH di Windows 10 hanya tersedia untuk Windows 10 dengan build number di atas 1804, untuk cek versi Windows 10, cari ‘run’ di search bar dan ketik ‘winver’ untuk melihat versi Windows 10 yang telah terpasang.

  • Jika versi Windows Anda merupakan versi 1804 atau di atasnya, Anda dapat menggunakan ‘Powershell’ atau ‘CMD’, yang dapat dicari melalui search bar.

  • Catatan: Perintah SSH belum tersedia untuk Windows di bawah versi Windows 10 1804. Untuk mengatasinya, SSH dapat diakses dengan client SSH seperti PuTTY.

  • Mengunduh PuTTY

    Untuk mengunduh PuTTY dapat diakses melalui link berikut:https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html, dan cari versi terbaru yang dapat dilihat di bagian 'Package files' di halaman tersebut, seperti yang ditandai dengan kotak biru pada di gambar berikut ini.

    Pastikan download sesuai Windows Anda (32-bit atau 64-bit). Untuk mengecek jenis Windows yang Anda gunakan, dapat melakukan hal berikut:

    1. Klik tombol start / windows dipojok kiri bawah dan buka kolom run [Windows key + r]
    2. Ketik dxdiag lalu tekan enter dan akan terlihat seperti gambar dibawah

  • Meng-install PuTTY

    • Setelah download selesai, buka folder dan klik Run

    • Pilih lokasi instalasi sesuai yang diinginkan

    • Dan klik Install

    • Setelah instalasi selesai, putty siap digunakan

  • Untuk Linux, dapat menggunakan program terminal pilihan seperti Gnome Terminal, Konsole, Tilix, dll

  • Untuk macOS, cukup gunakan program terminal bawaan dengan membukanya melalui Application List atau mencarinya dengan menggunakan Spotlight Search
